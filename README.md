# Expense Tracker
A simple and responsive expense tracker application that allows users to manage their daily finances by adding and viewing transactions.
## Features
- Add transaction with description and amount
- Add transaction button for easy entry
- Track income and expenses
- Clean and responsive design
- Transaction display list
- Delete transaction option
- Simple and user-friendly interface
